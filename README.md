# us-air-pollution-pyspark-ml-
this project implements an end-to-end Big Data pipeline for air quality analysis and prediction using Apache Spark and PySpark. The dataset is processed in a distributed consideration including batch processing, EDA, data preprocessing, Spark SQL manipulation, machine learning modeling.
Dataset source: https://www.kaggle.com/datasets/mexwell/us-air-pollution?resource=download 
Please download the dataset manually and place it in the `/data` directory before running the notebook.
